Project Mission

Traditional joysticks require hand coordination, but many patients with Quadriplegia or severe Spinal Cord Injuries retain only movement above the neck. We developed this to restore their independence using the tongue as a high-precision input device.

Target User Profile

  • Patients with total limb paralysis (C1-C4 spinal injury).
  • Individuals with Motor Neuron Disease (MND/ALS).
Wireless Protocol ESP-NOW (Low Latency)
Sensing Logic Capacitive Touch
Safety Feature Real-time Biometric Vitals Monitoring
